# MediAgenda 🏥 - Sistema de Gestión de Citas Médicas
*Universidad Estatal de Milagro (UNEMI)*  
Carrera de Ingeniería de Software | Introducción a la Ingeniería de Software

## 📝 Descripción del Proyecto
MediAgenda es una plataforma web responsiva diseñada para optimizar y automatizar el proceso de reserva, cancelación y control de citas médicas, mejorando la experiencia del paciente y la organización del personal de salud de la institución.

## 🛠️ Tecnologías y Requerimientos Destacados
* *Frontend:* HTML5, CSS3 (Diseño Responsivo con Media Queries) y JavaScript.
* *Backend:* Lógica en servidor con validación de duplicidad horaria (RF04).
* *Seguridad:* Cifrado de datos confidenciales de las historias clínicas (RNF03).
* *Rendimiento:* Respuestas optimizadas a menos de 3 segundos (RNF01).
